# Contributor Covenant Code of Conduct

> Adapted from [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## 我们的承诺

为了营造开放和友好的环境,我们作为贡献者和维护者承诺,无论年龄、体型、可见或不可见的残障、族裔、性别认同与表达、经验程度、教育、社会经济地位、国籍、个人外观、种族、种姓、肤色、宗教、性取向或性别认同,参与本项目的人员均能免于骚扰。

我们承诺以贡献于开放、欢迎、多元、包容、健康的社区的方式行事和互动。

## 我们的标准

有助于营造积极环境的行为示例:

- 表现出对他人的同理心和善意
- 尊重不同的意见、观点和经验
- 给予并优雅地接受建设性的反馈
- 对错误负责,向受影响者道歉,并从经历中学习
- 关注的不仅是个人,而是整个社区的最大利益

不可接受的行为示例:

- 使用性化语言或图像,以及任何形式的性关注或性挑逗
- 挑衅、侮辱或贬损性的评论,以及个人或政治攻击
- 公开或私下的骚扰
- 未经明确许可发布他人的私人信息(例如物理或电子邮件地址)
- 在专业环境中可被合理认定为不当的其他行为

## 维护者职责

社区维护者负责澄清和执行可接受行为的标准,并将对任何被认为不当、威胁、冒犯或有害的行为采取适当且公平的纠正措施。

## 适用范围

本行为准则适用于所有社区空间,以及当个人在公共空间正式代表社区时。代表本社区的例子包括使用官方电子邮件地址、在官方社交媒体账户发布内容,或作为指定代表参加线上或线下活动。

## 执行

辱骂、骚扰或其他不可接受行为可向社区维护者报告,联系方式见 [`SECURITY.md`](SECURITY.md) 中的 security email。所有投诉将被及时和公正地审查与调查。

所有社区维护者有义务尊重报告者的隐私和安全。

## 执行准则

社区维护者将遵循以下社区影响准则,以判定其认为违反本行为准则的任何行为的后果:

### 1. 更正

**社区影响**:使用不当语言或在社区中被认为不专业或不受欢迎的其他行为。

**后果**:私下书面警告。可能要求公开道歉。

### 2. 警告

**社区影响**:通过单一事件或一系列行为构成的违规。

**后果**:警告,如继续违规将面临进一步行动。在指定时间内不得与相关人员互动,包括在社区空间和外部渠道。违反这些条款可能导致临时或永久封禁。

### 3. 临时封禁

**社区影响**:严重违反社区标准,包括持续的不当行为。

**后果**:在指定时间内禁止与社区进行任何形式的互动或公开交流。在此期间,不允许与相关人员进行任何公开或私下的互动。违反这些条款可能导致永久封禁。

### 4. 永久封禁

**社区影响**:表现出违反社区标准的模式,包括持续的不当行为、骚扰个人或对个人的攻击或贬低。

**后果**:永久禁止任何形式的社区公开互动。
